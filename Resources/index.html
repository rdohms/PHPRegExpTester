PHP Regular Expression Tester

This Application is an experiment in the use of Appcelerator Titanium with a PHP backend. It comes to aid developers who wish to tweak their Regular Expressions without having to setup a testing environment or a loose file for that, allowing them to run regular expressions against the desired text.
